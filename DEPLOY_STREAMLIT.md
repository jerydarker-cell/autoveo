# Deploy AUTO VEO Studio v2.1 lên Streamlit

1. Push repo lên GitHub.
2. Vào Streamlit Community Cloud.
3. New app → chọn repo.
4. Main file: `app.py`.
5. Deploy.

Có `packages.txt` để cài FFmpeg trên cloud.
